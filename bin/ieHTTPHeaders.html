What is ieHTTPHeaders?

It is an explorer bar for Microsoft Internet Explorer 6 and later, that displays the HTTP request and response headers for you as you use the browser and surf.

Troubleshooting

I've had occasional reports about installation problems, where no explorer bar option for ieHTTPHeaders is available. It seems that it is best fixed by re-registering the explorer bar. To do so, run 'regsvr32 iehttptrace.dll' from directory where ieHTTPHeaders is installed.
